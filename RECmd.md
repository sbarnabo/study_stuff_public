# RECmd 
## The command-line implementation of Registry Explorer.  
### Considered an "advanced" tool.  

Batch examples tell the tool what keys and values to extract.  
UserActivity.reb is an example.  Use Notepad++ or any other editor to view.  

### Example Commandline:  
- cd /d C:\<program_location>\ZimmermanTools\net9\RECmd
- RECmd.exe -f source_location\NTUSER.DAT_clean --bn .\BatchExamples\UserActivity.reb --csv output_location
